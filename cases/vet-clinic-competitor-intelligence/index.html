Competitive Intelligence

Vet Clinic Competitor Intelligence Tool

One-click competitive landscape dashboard: input a clinic website, get an interactive map + comparison of all competitors, reviews, and ad presence—live on sales calls.

The Challenge

A SaaS company selling practice management software to veterinary clinics wanted to differentiate on sales calls by showing prospects exactly how they stack up against nearby competitors.

Sales reps were manually Googling competitors, pulling reviews, checking Facebook Ads Library, and creating rough comparison slides—taking 30-45 minutes per prospect. By the time the research was done, the call had already happened.

They needed:

  • Instant competitor analysis during live sales calls
  • Interactive map showing all competitors within 20-mile radius
  • Review comparison (Google, Yelp, Facebook)
  • Ad presence detection (Google Ads, Facebook Ads)
  • Shareable link to send prospects after the call

The Solution

AutoFlux built a real-time competitive intelligence tool using n8n for data orchestration, Apify for scraping, Google Maps API for location data, and Lovable for the interactive dashboard:

How It Works:

  1. Input:
    • Sales rep enters clinic website URL into simple form
    • System extracts business name + location from website
  2. Competitor Discovery (Google Maps API):
    • Searches Google Maps for "veterinary clinics" within 20-mile radius
    • Pulls: name, address, phone, rating, review count, website
    • Returns 10-30 competitors depending on market density
  3. Review Scraping (Apify):
    • Scrapes Google, Yelp, and Facebook reviews for all competitors
    • Calculates average ratings, total reviews, sentiment trends
    • Identifies top review themes (pricing, customer service, wait times)
  4. Ad Presence Check:
    • Checks Google Ads for search term "vet clinic [city]"
    • Scrapes Facebook Ads Library for active ad campaigns
    • Flags which competitors are advertising (and estimated spend)
  5. Dashboard Generation (Lovable):
    • Creates interactive map with all competitors plotted (color-coded by rating)
    • Comparison table: rating, reviews, ad presence, website quality
    • Shareable link generated instantly (unique URL per prospect)
  6. Live on Sales Calls:
    • Rep shares screen during call to show prospect their competitive landscape
    • Uses data to position product ("I see 3 of your competitors are running ads—here's how we help you compete")

Tech Stack:

n8n (orchestration) Lovable (dashboard UI) Google Maps API Apify (review scraping) Facebook Ads Library API Custom web scrapers
