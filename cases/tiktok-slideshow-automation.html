Creative Automation

TikTok Slideshow Automation Platform

AI-powered creative production that scrapes viral TikTok slideshows, analyzes patterns with GPT, and generates multiple text/image variants—cutting campaign time by 75%.

The Challenge

Our client, an e-commerce brand running high-volume TikTok ad campaigns, was spending 60+ minutes per campaign manually researching viral slideshow trends, downloading assets, and creating variations for A/B testing.

Their creative team was bottlenecked—unable to scale ad production without hiring more designers. They needed a way to automate the research, extraction, and variant generation process while maintaining creative quality.

The Solution

AutoFlux built a fully automated TikTok creative production platform using n8n, OpenAI GPT-4, and custom computer vision logic:

How It Works:

  1. Viral Content Scraping: Automatically scrapes TikTok for top-performing slideshow ads in target niches using TikTok API + Apify.
  2. Pattern Analysis (GPT-4): Sends scraped content to GPT-4 to analyze:
    • Winning text hooks and messaging patterns
    • Image composition and visual themes
    • Timing, pacing, and transitions
  3. Variant Generation: Creates 5-10 slideshow variants per viral template:
    • Generates multiple text overlays (different hooks, CTAs)
    • Applies smart text placement rules using computer vision logic
    • Outputs different product/image combinations
  4. Asset Delivery: Automatically organizes and delivers all variants to Google Drive, ready for the media buyer to launch.

Tech Stack:

n8n (workflow orchestration) OpenAI GPT-4 (pattern analysis) TikTok API Apify (web scraping) Google Drive API Custom CV Logic
